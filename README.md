# Medical Database SQL Practice

의료데이터베이스 수업에서 작성한 SQL 실습 코드를 정리한 저장소입니다.  
기본 SQL 문법부터 집계, 조건문, 서브쿼리, 사용자 정의 함수, JOIN, UNION, 제약조건까지 단계적으로 정리했습니다.

## Contents

| File | Description |
|---|---|
| `01_basic_table_and_query_review.sql` | 테이블 생성, 데이터 삽입, 기본 SELECT, ALTER, DROP 등 SQL 기초 복습 |
| `02_sorting_window_string_functions.sql` | 정렬, 순위 함수, 문자열 함수 실습 |
| `03_groupby_having_aggregate_queries.sql` | GROUP BY, HAVING, COUNT, AVG, MIN, MAX 등 집계 쿼리 |
| `04_conditional_logic_and_subqueries.sql` | IF, CASE, 서브쿼리, ANY, ALL, EXISTS 실습 |
| `05_user_defined_functions.sql` | MySQL 사용자 정의 함수 생성 및 활용 |
| `06_join_and_set_operations_examples.sql` | INNER JOIN, LEFT JOIN, RIGHT JOIN, SELF JOIN, UNION 예제 |
| `07_join_union_practice_queries.sql` | JOIN과 UNION을 활용한 실습 쿼리 |
| `08_ddl_dml_constraints.sql` | UPDATE, DELETE, ALTER TABLE, PRIMARY KEY, FOREIGN KEY, CASCADE 제약조건 실습 |

## Main Topics

- Basic SQL queries
- Data filtering and sorting
- Aggregate functions
- Conditional logic
- Subqueries
- User-defined functions
- JOIN operations
- UNION / UNION ALL
- Table constraints
- DDL and DML practice

## Note

이 저장소의 SQL 코드는 의료데이터베이스 수업 실습 내용을 기반으로 정리한 것입니다.  
공개 저장소 업로드를 위해 예시 데이터와 환자명은 익명화 또는 실습용 데이터 형태로 정리하는 것을 권장합니다.
